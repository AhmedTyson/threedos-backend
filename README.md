# threedos-backend

This is my personal task delivery repository for my work as a Backend Development Delegate at **ThreeDOS** — a student-led organization operating as a simulated real-world tech company, focused on developing market-ready engineering talent.

Every folder under `tasks/` corresponds to a single session. Each session contains the task brief and its associated solution or documentation. Nothing more, nothing is carried here that does not belong to a submitted task.

---

## <img src="https://api.iconify.design/lucide:folder-tree.svg?color=%238A2BE2" width="24" align="top" /> Structure

```
threedos-backend/
│
├── README.md
├── .gitignore
│
└── tasks/
    ├── task-01/
    │   ├── task.md
    │   ├── diagrams/
    │   └── backend/
    ├── task-02/
    │   ├── task.md
    │   ├── diagrams/
    │   └── backend/
    └── ...
```

Each `task-XX` folder is self-contained. The `task.md` file describes the assignment. The solution file(s) hold the actual work — a script, a schema, a documented breakdown, or any combination the task required.
